# OpenNOW

OpenNOW is an open-source GeForce NOW client.

> [!IMPORTANT]
> This repository is now for the web-based version only. If you are looking for the native client, please visit **[OpenNOW Streamer](https://github.com/OpenNOW-Project/Streamer)**.

## About
OpenNOW is a custom GeForce NOW client created by reverse engineering the official NVIDIA client. It provides a clean, open-source way to access GeForce NOW from your browser.

## Getting Started
To run the development server:
```bash
bun install
bun run dev
```
